# Screenshot Instructions

To add screenshots to the README:

1. **Dashboard Screenshot** (`dashboard-weekly-overview.png`)
   - Take a screenshot of the weekly calendar view showing:
     - Subject filters
     - Weekly overview with colored session blocks
     - Daily totals
     - Summary statistics

2. **Ranking System Screenshot** (`ranking-system.png`)
   - Take a screenshot showing:
     - The 5-tier ranking system (Baus → Sherm → Squid → French Mouse → Taus)
     - Rank icons and descriptions
     - Weekly focus time requirements

3. **Save the images** in this directory (`docs/images/`) with the exact filenames:
   - `dashboard-weekly-overview.png`
   - `ranking-system.png`

4. The README.md will automatically display them once added.

## Optional Additional Screenshots

You can also add:
- `leaderboard.png` - Global or friends leaderboard view
- `friends-page.png` - Friend management interface
- `profile-page.png` - User profile view

Then update the README.md to include them in the Screenshots section.
